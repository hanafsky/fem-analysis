# ベンチマークで与えたプロンプト

Claude Opus 4.6 / 4.7 に対して、`run-4-6/` と `run-4-7/` でそれぞれ実行した際に
与えたプロンプトの全文です。両モデルとも完全に同じ文面を投げています。

---

2つの金属ブロック（銅と鋼）が不完全接触している状況の
非定常熱伝導解析をFreeFEM++で実施して、PyVistaで可視化してください。

【問題設定】
- 形状: 左ブロック(銅, 50mm×30mm×20mm)と右ブロック(鋼, 50mm×30mm×20mm)が
  x方向に接触。接触面は x=50mm にある
- 物性値:
  - 銅: λ=400 W/mK, ρ=8960 kg/m³, Cp=385 J/kgK
  - 鋼: λ=50 W/mK, ρ=7850 kg/m³, Cp=490 J/kgK
- 接触熱抵抗: Rc = 1e-4 m²K/W (接触面での温度跳び)
- 初期条件: 全体 T=293K
- 境界条件:
  - 左端(x=0): T=373K (固定)
  - 右端(x=100mm): 対流 h=10 W/m²K, T_inf=293K
  - その他の面: 断熱
- 解析時間: 0〜300秒、Δt=5秒

【成果物】
1. FreeFEM++コード (.edp)
2. 実行して得られた温度場データ
3. PyVistaによる可視化 (t=60, 150, 300秒の3断面)
4. 接触面での温度ジャンプがRc=1e-4 m²K/Wに整合しているかの検証レポート

【制約】
- 2D問題として解いて構いません(xy断面、厚み方向は無視)
- 接触面の温度不連続を適切に扱ってください(単一メッシュで平均化しないこと)
