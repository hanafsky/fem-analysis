# 片持ち梁の解析

## プロンプト例

> 鋼製の片持ち梁（1m x 0.1m）を解析してください。
> 左端を固定、右端に 1 MPa の分布荷重をかけます。
> Von Mises 応力と変形形状を表示してください。

## 期待される動作

1. Claude が平面応力の定式化で `cantilever.edp` を生成（E=210 GPa, nu=0.3）
2. `FreeFem++ cantilever.edp` を実行
3. メッシュ（`mesh.msh`）、変位（`u1.txt`, `u2.txt`）、応力（`vonmises.txt`）を出力
4. PyVista で変形メッシュに Von Mises 応力コンターを重ねて可視化
5. 自己評価：変形方向、固定端の応力集中、境界条件の確認
6. 最大変位・最大応力の数値とともに最終画像を提示

## 技術ポイント

- 平面応力の仮定（薄板、sigma_zz = 0）
- P2 要素で変位を高精度計算、出力時に P1 へ射影
- 変形倍率は自動スケール（概ね 0.1 * L / 最大変位）
- Von Mises 応力は固定端（左端）付近で最大になるはず
- 解析解との比較：先端たわみ delta = PL^3 / (3EI)

## プロンプトのバリエーション

**穴あき：**
> 同じ片持ち梁ですが、中央に半径 0.02m の円孔を追加してください。
> 孔周りの応力集中を確認したいです。

**材料変更：**
> アルミ（E=70 GPa, nu=0.33）の片持ち梁で同じ解析をしてください。
> 鋼との変位の違いを比較してください。

**自重あり：**
> 片持ち梁に自重（密度 7800 kg/m^3）も加えてください。
